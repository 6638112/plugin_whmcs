PAYSSION Payment Gateway Module for WHMCS
=================

<h4>Installation</h4>
1. Upload files to your WHMCS installation keeping folder structure
2. Login to WHMCS as Administrator
3. Go to Setup > Payments > Payment Gateways in the main menu
4. Select "All payment gateways" tab and clock on PAYSSION
5. Fill in API Key and Secret Key which you can find at your Payssion account.
6. Save changes.
